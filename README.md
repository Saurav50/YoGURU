## Run Locally

Clone the project

```bash
  git clone https://github.com/Saurav50/YoGURU.git
```

Go to the project directory

```bash
  cd YoGURU
```

Install dependencies

```bash
  pip install notebook
```

Start the server

```bash
  Run
```
